Este proyecto ejecuta una pagina web para la simulación de un sitio web dude venta de libros esta pagina es creada con html, utilizando una base de datos generada en MySQ. 
Se implementan servelts para poder manejar la peticiones de la pagina en un script en java que funciona como el controlador en el modelo MVC y también realiza las peticiones a la base de datos para el despliegue de información.
